## 🚀 Tecnologias Utilizadas
**Backend:** PHP 8
**Banco de Dados:** MySQL (phpMyAdmin)
**Frontend:** CSS

## 🔧 Como rodar o projeto localmente
1. Baixe a pasta para dentro do seu xampp/htdocs.
2. Abra o phpMyAdmin e crie um banco de dados chamado techos_db.
3. Importe o arquivo .sql que está dentro da pasta database/.
4. Acesse no navegador: http://localhost/TechOs.
